# Phishing Drill Evidence Console

## Summary
Phishing Drill Evidence Console is a full-stack workspace for managing phishing drill workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Phishing Drill Evidence Console addresses a real-world gap in security operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Phishing Drill Evidence Console control dashboard
- Phishing Drill Evidence Console issue board
- Phishing Drill Evidence Console evidence review views

## Backend Features
- Phishing Drill Evidence Console security APIs
- Phishing Drill Evidence Console risk endpoints
- Phishing Drill Evidence Console audit summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
