# Court Deadline Guard

## Summary
Court Deadline Guard is a full-stack workspace for managing court deadline workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Court Deadline Guard addresses a real-world gap in deadline tracking, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Court Deadline Guard calendar dashboard
- Court Deadline Guard deadline board
- Court Deadline Guard checklist views

## Backend Features
- Court Deadline Guard case APIs
- Court Deadline Guard deadline workflows
- Court Deadline Guard reminder endpoints

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
