# Waste Pickup Route Console

## Summary
Waste Pickup Route Console is a full-stack workspace for managing waste pickup workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Waste Pickup Route Console addresses a real-world gap in city coordination, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Waste Pickup Route Console operations map
- Waste Pickup Route Console route board
- Waste Pickup Route Console issue dashboard

## Backend Features
- Waste Pickup Route Console city service APIs
- Waste Pickup Route Console routing endpoints
- Waste Pickup Route Console trend metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
