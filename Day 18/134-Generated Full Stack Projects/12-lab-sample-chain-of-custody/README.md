# Lab Sample Chain-of-Custody

## Summary
Lab Sample Chain-of-Custody is a full-stack workspace for managing lab sample workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Lab Sample Chain-of-Custody addresses a real-world gap in sample traceability, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Lab Sample ChainofCustody inventory views
- Lab Sample ChainofCustody custody timeline
- Lab Sample ChainofCustody storage status cards

## Backend Features
- Lab Sample ChainofCustody sample APIs
- Lab Sample ChainofCustody transfer workflows
- Lab Sample ChainofCustody audit trails

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
