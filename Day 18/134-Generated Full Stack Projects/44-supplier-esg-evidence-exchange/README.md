# Supplier ESG Evidence Exchange

## Summary
Supplier ESG Evidence Exchange is a full-stack workspace for managing supplier esg workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Supplier ESG Evidence Exchange addresses a real-world gap in evidence exchange, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Supplier ESG Evidence Exchange supplier portal
- Supplier ESG Evidence Exchange scorecard dashboard
- Supplier ESG Evidence Exchange expiry views

## Backend Features
- Supplier ESG Evidence Exchange evidence APIs
- Supplier ESG Evidence Exchange review endpoints
- Supplier ESG Evidence Exchange history logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
