# Procurement Bid Comparison Engine

## Summary
Procurement Bid Comparison Engine is a full-stack workspace for managing procurement bid workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Procurement Bid Comparison Engine addresses a real-world gap in supplier evaluation, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Procurement Bid Comparison Engine comparison matrix
- Procurement Bid Comparison Engine weighted scorecards
- Procurement Bid Comparison Engine award board

## Backend Features
- Procurement Bid Comparison Engine bid APIs
- Procurement Bid Comparison Engine scoring endpoints
- Procurement Bid Comparison Engine decision logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
