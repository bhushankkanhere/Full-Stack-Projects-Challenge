# Broadband Outage Reporting Network

## Summary
Broadband Outage Reporting Network is a full-stack workspace for managing broadband outage workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Broadband Outage Reporting Network addresses a real-world gap in outage coordination, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Broadband Outage Reporting Network outage map
- Broadband Outage Reporting Network report form
- Broadband Outage Reporting Network restoration board

## Backend Features
- Broadband Outage Reporting Network outage APIs
- Broadband Outage Reporting Network clustering stubs
- Broadband Outage Reporting Network alert summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
