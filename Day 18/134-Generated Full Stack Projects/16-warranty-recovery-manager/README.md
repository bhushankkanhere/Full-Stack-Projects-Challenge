# Warranty Recovery Manager

## Summary
Warranty Recovery Manager is a full-stack workspace for managing warranty recovery workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Warranty Recovery Manager addresses a real-world gap in recovery management, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Warranty Recovery Manager claim dashboard
- Warranty Recovery Manager vendor status board
- Warranty Recovery Manager serial lookup views

## Backend Features
- Warranty Recovery Manager claim APIs
- Warranty Recovery Manager recovery workflows
- Warranty Recovery Manager reimbursement summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
