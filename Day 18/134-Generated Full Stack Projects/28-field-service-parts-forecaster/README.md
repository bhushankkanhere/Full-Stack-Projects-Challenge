# Field Service Parts Forecaster

## Summary
Field Service Parts Forecaster is a full-stack workspace for managing field service workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Field Service Parts Forecaster addresses a real-world gap in forecasting operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Field Service Parts Forecaster forecast dashboard
- Field Service Parts Forecaster inventory board
- Field Service Parts Forecaster exception views

## Backend Features
- Field Service Parts Forecaster forecast APIs
- Field Service Parts Forecaster demand endpoints
- Field Service Parts Forecaster threshold alerts

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
