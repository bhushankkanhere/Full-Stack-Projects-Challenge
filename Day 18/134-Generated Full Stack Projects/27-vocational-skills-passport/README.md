# Vocational Skills Passport

## Summary
Vocational Skills Passport is a full-stack workspace for managing vocational skills workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Vocational Skills Passport addresses a real-world gap in skills verification, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Vocational Skills Passport credential wallet
- Vocational Skills Passport assessment board
- Vocational Skills Passport employer verification views

## Backend Features
- Vocational Skills Passport credential APIs
- Vocational Skills Passport verification endpoints
- Vocational Skills Passport assessment logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
